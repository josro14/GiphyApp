//Welcome to my Cool Giphy App!

Start by trying to click any of the blue buttons at the bottom to generate a set of GIFs related to the button title. 

You can also generate new buttons for different categories of GIFs by searching GIF titles in the search bar and then pressing enter or clicking the "New Button" button. 

That button will appear among the button set at the bottom which can now be clicked to generate new sets of GIFs.

Try searching GIFs about reactions or emotions. Then if you get bored, try searching any kind of GIFs you'd like!
